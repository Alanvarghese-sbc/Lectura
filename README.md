## Problem Statement
[cite_start]Students often struggle to listen and take notes simultaneously during lectures, leading to missed key points[cite: 29]. [cite_start]This project provides an AI solution that converts spoken lectures into clear study notes, quizzes, and flashcards[cite: 30].

## Tech Stack
- **Python**: Core logic.
- [cite_start]**Whisper (OpenAI)**: For Speech-to-Text transcription[cite: 30].
- [cite_start]**BART (Hugging Face)**: For high-quality text summarization[cite: 30].
- [cite_start]**T5 (Google)**: For generating quizzes and study aids[cite: 3].
- **Streamlit**: For the interactive web dashboard.

## Features
1. **Automated Transcription**: Converts raw audio to text.
2. **Summarization**: Generates short, actionable study notes.
3. **Active Recall**: Creates quizzes and flashcards automatically.
